# 🚀 Python Port Scanner: Evolution | Evolución

This project showcases the development of a network reconnaissance tool, evolving from a basic functional script to a professional Object-Oriented Programming (OOP) implementation.
*Este proyecto muestra el desarrollo de una herramienta de reconocimiento de red, evolucionando desde un script funcional básico hasta una implementación profesional orientada a objetos (POO).*

---

# 🛠️ Phase 1: Simple Scanner | Fase 1: Escáner Simple
The initial version uses the `socket` library to verify connectivity with specific ports of a domain (e.g., `google.com`).
*La versión inicial utiliza la biblioteca `socket` para verificar la conectividad con puertos específicos de un dominio (ej. `google.com`).*

# 🛡️ Phase 2: Scanner Pro (OOP Version) | Fase 2: Escáner Pro (Versión POO)
In this update, the script was refactored using **Object-Oriented Programming**, improving scalability and data handling.
*En esta actualización, el script fue refactorizado utilizando **Programación Orientada a Objetos**, mejorando la escalabilidad y el manejo de datos.*

# Key Features | Características Destacadas:
* **Class Logic / Lógica de Clases:** Modular and reusable structure. / *Estructura modular y reutilizable.*
* **Datetime Module / Módulo Datetime:** Precise execution time logging. / *Registro preciso del tiempo de ejecución.*
* **Data Persistence / Persistencia de Datos:** Automatic `.txt` report generation. / *Generación automática de reportes `.txt`.*
* **Append Mode / Modo Append (`a`):** Cumulative history that does not overwrite previous results. / *Historial acumulativo que no sobreescribe resultados anteriores.*
