# Python Port Scanner: Evolution

Este proyecto muestra el desarrollo de una herramienta de reconocimiento de red, evolucionando desde un script funcional básico hasta una implementación orientada a objetos (POO).

---

# Fase 1: Simple Scanner
La versión inicial utiliza la biblioteca `socket` para verificar la conectividad con puertos específicos de un dominio (ej. `google.com`).

#  Fase 2: Scanner Pro (OOP Version)
En esta actualización, el script fue refactorizado utilizando **Programación Orientada a Objetos**, mejorando la escalabilidad y el manejo de datos.

# Características Destacadas:
* **Lógica de Clases:** Estructura modular y reutilizable.
* **Módulo Datetime:** Registro preciso del tiempo de ejecución.
* **Persistencia de Datos:** Generación automática de reportes `.txt`.
* **Modo Append (`a`):** Historial acumulativo que no sobreescribe resultados anteriores.
