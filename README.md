SCANNER PRO:

Esta es la versión actualizada el programa, agregué clases y métodos generales de python. Además, el uso del "datetime" para que en la impresión nos de el tiempo en el que tardó en hacerse el escaneo. Por último, el resultado del código en la terminal generará un programa txt que guarde los resultados; con el arreglo "a" (append), no borrará los anteriores, sino que guardará uno por uno.
